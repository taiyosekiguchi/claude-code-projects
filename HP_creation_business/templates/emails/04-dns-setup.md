## 注意事項

### 他のドメインサービスの場合

**ムームードメイン**:
```
レコードタイプ: A
サブドメイン: 空欄
値: 76.76.21.21
```

**Freenom**:
```
Type: A
Name: 空欄
Target: 76.76.21.21
TTL: 14400
```

### サブドメイン（www）も設定する場合

```
レコードタイプ: CNAME
ホスト名: www
値: cname.vercel-dns.com
```
