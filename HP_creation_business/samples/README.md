## 🚀 プレビュー方法

### 方法1: ブラウザで直接開く（最も簡単）

1. 各テーマのフォルダに移動
2. `index.html` をダブルクリック
3. ブラウザで表示される

### 方法2: ローカルサーバーで確認（推奨）

Python3を使用:

```bash
# テーマAをプレビュー
cd appricot-theme-a
python3 -m http.server 8000
# http://localhost:8000 にアクセス

# テーマBをプレビュー
cd appricot-theme-b
python3 -m http.server 8001
# http://localhost:8001 にアクセス

# テーマCをプレビュー
cd appricot-theme-c
python3 -m http.server 8002
# http://localhost:8002 にアクセス
```

Node.jsを使用:

```bash
# http-server をインストール（初回のみ）
npm install -g http-server

# 各テーマフォルダで実行
cd appricot-theme-a
http-server -p 8000
```

## 🔧 カスタマイズ方法

### 色の変更

各テーマの `style.css` の `:root` セクションでカラーを変更できます：

```css
:root {
    --color-primary: #FF6B35;  /* メインカラー */
    --color-text: #1a1a1a;     /* テキスト色 */
    /* ... */
}
```

### フォントの変更

`<head>` 内のGoogle Fontsのリンクと、CSSの `font-family` を変更：

```html
<link href="https://fonts.googleapis.com/css2?family=別のフォント&display=swap" rel="stylesheet">
```

### コンテンツの変更

HTMLファイル内の以下を変更：
- 会社名
- 代表者名
- 所在地
- 事業内容
- メールアドレス

## 📝 注意事項

### 問い合わせフォームについて

現在のサンプルには**問い合わせフォームは含まれていません**。

理由:
- 最低限の企業HPとしては不要
- 追加開発として、オプションで提供可能

追加したい場合:
- Formspree（無料）を使用
- または、Googleフォーム埋め込み

### 電話番号について

サンプルには電話番号欄がありません（顧客の要望による）。

追加したい場合:
- お問い合わせセクションに追加
- 会社概要の項目として追加
